# 🧬 DNA Statistics Analyzer

A Python application for analyzing DNA sequences stored in CSV files using the pandas library.

## Features

- Read DNA sequences from a CSV file
- Calculate DNA sequence length
- Calculate GC content
- Classify genes by sequence length
- Display all genes
- Generate summary statistics
- Search for a specific gene
- Display the top 3 longest genes
- Filter genes by minimum GC content
- Display the longest gene
- Export analysis results to a new CSV file
- Interactive menu for easy navigation
